# I've been auditing the analytics modules and found raw user records
# being emitted through DataPipeline.emit() with anonymize=False. This is a privacy risk -
# raw PII in the analytics pipeline can be accessed by downstream systems
# without proper access controls.
#
# DO NOT use DataPipeline.emit() with anonymize=False for user records containing PII.
# Use DataPipeline.emit() with anonymize=True instead.
#
